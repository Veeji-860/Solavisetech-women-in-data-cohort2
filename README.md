# Solavisetech-women-in-data-cohort2
1. Introduce the program/project  
The program is a Data Science training track that takes you through the full analytics workflow — from data collection and cleaning with Python and SQL, to exploratory analysis, machine learning, and communicating results to stakeholders. It’s project-based, using real datasets so you graduate with both skills and portfolio work.

2. Briefly introduce yourself
I’m Nana, and I hold a degree in Business Administration and an AWS Solutions Architect certification. My background has given me a strong foundation in business processes, cloud infrastructure, and systems thinking. I’m comfortable working with technical tools and translating technical outcomes into business value. Now I want to add data analysis and machine learning to that skill set so I can solve problems end-to-end.

3. Write your expectations from the program
I expect a hands-on, structured program that pushes me beyond tutorials into working with messy, real-world data. I expect to learn industry-standard workflows, get direct feedback on my analysis and code, and understand how data science integrates with cloud infrastructure — especially on AWS. I also expect the training to bridge the gap between technical modeling and business decision-making.

4. Mention what you hope to learn during the training 
- Technical skills: Python for data analysis, Pandas, SQL for data extraction, and data visualization with Matplotlib/Seaborn.
- Machine learning: Core algorithms for regression, classification, and clustering, plus proper model evaluation and validation techniques.  
- Cloud + Data: How to build scalable data pipelines and run analytics workflows on AWS, tying into my Solutions Architect background.  
- Business application: How to frame business problems as data problems, and present insights in a way that drives action for non-technical stakeholders.
